Information

Technology and development details are outlined in technology page. This website is built and maintained by Yalim Aytul.

Since 1998 many paintings are photographed and added to this website but not much content is changed. Even home page still has the same text from the very beginning. After all this website is for displaying art work and not talking about it. However, technology powering this website has constantly been improved. Below is a time line of significant changes on yalimaytul.com over the years.

  • 1998

    yalimaytul.com is born!

    It's just few html pages with static content and few paintings.

  • 1999

    Most paintings are photographed

    Most of the paintings are photographed and published. Site is built with perl.

  • 2000

    Hello PHP

    Functionality enhanced, reviews are added. Site migrated to php.

  • 2001

    A new look!

    Site look and feel has changed significantly, this will be the new design for yalimaytul.com until 2014.

  • 2002-2006

    New paintings photographed

    More paintings are photographed and added. Site is overhauled for SEO. Started using CodeIgniter.

  • 2007-2010

    Mobile website

    From initial idea to finalization mobile friendly site created. New galleries are added.

  • 2011

    SEO changes

    Lots of structural changes for SEO and mobile.

  • 2012

    Hello Responsive!

    Mobile site replaced by responsive design. Hosting changed.

  • 2013

    Hosted by OpenShift

    Hosting changed again! Site is hosted on OpenShift platform. Now using AngularJS and database changed to MongoDB.

  • 2014

    New design!

    Site completely redesigned. Fully responsive. Build, testing, publishing tasks are now done with Grunt!

  • 2015

    Going ReSTful!

    yalimaytul.com Painting API now powers the paintings site. Most images are responsive and HD. Site featured in OpenShift developer spotlight.

  • 2016

    Rebuilt with Spring Boot

    5th generation is built with Java and Spring Boot.

  • 2017

    6th generation

    6th generation is built with Java and Spring Boot.

  • 2020

    Resume and API websites

    Resume and API have their dedicated sites.
